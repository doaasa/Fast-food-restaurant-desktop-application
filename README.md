# Fast Food Restaurant Desktop Application
### This is a basic desktop application built with C# .NET that allows users to create, print, and reset receipts for a fast food restaurant.
<img src="https://github.com/doaasa/Fast-food-restaurant-desktop-application/blob/main/Photos/main%20page.png" width=100%> 

<br>
<br>

# Buttons in Application:
## 1- Reset Button : to reset the receipt and the checkboxes.
## 2- ADD Button : to add the items to receipt , calculate subtotal , calculate tax(0.05) and calculate Grand Total.
## 3- Print Button : to print the receipt that show the print dialog with the receipt items.

# Technical Options:
## 1- Date label : to show the current date and time using timer tool.
## 2- Close Button : to close the application.
## 3- Checkboxes : to choose the item that wanted to add to receipt.
## 4- Textboxes : to add the count of each item before add it to receipt.
